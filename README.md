# Pavel & Darina Wedding Invitation

Современный адаптивный сайт-приглашение на свадьбу, разработанный с использованием React, TanStack Start, Vite и Tailwind CSS.

## ✨ Возможности

- Современный минималистичный дизайн
- Таймер обратного отсчёта до начала мероприятия
- Программа свадебного дня
- Раздел Dress Code
- Информация о месте проведения
- RSVP-форма с поддержкой Netlify Forms
- Полностью адаптивный интерфейс для мобильных устройств и компьютеров
- Плавные анимации и современные визуальные эффекты

---

## 🛠️ Технологии

- React 19
- TypeScript
- Vite
- TanStack Start
- TanStack Router
- Tailwind CSS 4
- Lucide React

## 🚀 Начало работы

### Установка зависимостей

```bash
npm install
```

### Запуск проекта

```bash
npm run dev
```
После запуска сайт будет доступен по адресу:

```
http://localhost:3000
```

или адресу, который укажет Vite.
